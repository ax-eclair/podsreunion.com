# AirPods Case Images

These files support the PodsReunion AirPods case taxonomy.

Keep the model number at the start of each filename. The filename is part of the product identification workflow.

## Folders

- `web/` contains the less glossy images intended for product UI usage.
- `original/` contains the untouched source images.

The `web/` filenames intentionally do not include `-reworked`; future code should use these paths by default.

## Files

- `A1602.png` - Lightning Charging Case for AirPods 1 / 2
- `A1938.png` - Wireless Charging Case for AirPods 1 / 2
- `A2566-magsafe.png` - MagSafe Charging Case for AirPods 3
- `A2897-lightning.png` - Lightning Charging Case for AirPods 3
- `A3058.png` - Charging Case for AirPods 4
- `A3059-ANC.png` - Wireless Charging Case for AirPods 4 with ANC
- `A2190.png` - Charging Case / MagSafe Charging Case for AirPods Pro 1
- `A2700-magsafe-lightning.png` - MagSafe Charging Case for AirPods Pro 2, Lightning
- `A2968-magsafe-lightning.png` - MagSafe Charging Case for AirPods Pro 2, USB-C; filename says Lightning, but model A2968 is USB-C.
- `A3122-magsafe.png` - MagSafe Charging Case for AirPods Pro 3, USB-C
- `A1290-magsafe.png` - Duplicate of `A2190.png`; likely a filename typo to verify.

## Usage Note

Treat these as reference assets until rights are confirmed. For the public website, prefer original renders, licensed 3D models, or custom illustrations.
