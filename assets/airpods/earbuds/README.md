# AirPods Earbud Images

Each model folder is generated from the cleaner `assets/airpods/products/web/*-earbuds.png` source image, not from screenshots.

Each model folder contains:

- `og.png` - the product web source image used for extraction.
- `right.png` - one cropped earbud extracted from the source image for UI usage.
- `left.png` - horizontal mirror of `right.png` for left-side presentation.

The side images are extracted from the higher-quality product web assets so they avoid screenshot compression. If a model attribution is wrong, rename the model folder and update this README.

AirPods 1 and AirPods 2 side images use the rear-view earbud from the source pair and are lightly rotated so the stem reads vertically in the UI.
