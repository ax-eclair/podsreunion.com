# AirPods Product Images

These files support the PodsReunion AirPods product taxonomy.

Keep filenames lowercase and product-led:

- `airpods-{generation}-earbuds.png` for earbuds-only product renders.
- `airpods-{generation}-with-case.png` for open charging-case renders.
- Add a view suffix, such as `-front` or `-side`, when a product has multiple case views.

## Folders

- `web/` contains the less glossy images intended for product UI usage.
- `original/` contains the untouched source images.
- `comparisons/` contains review-only comparison sheets.

The `web/` filenames intentionally do not include `-reworked`; future code should use these paths by default.
